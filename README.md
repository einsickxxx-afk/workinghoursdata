# 해양사업부 근무실적 현황

주간 근무시간 확정 데이터를 불러와 52시간 초과 근무 현황, 위험군, 부서·사업장별 평균 근무시간 등을 한눈에 보여주는 단일 HTML 대시보드입니다. 별도 서버나 프로그램 설치 없이 브라우저에서 파일을 열기만 하면 동작합니다.

## 빠른 시작

1. `해양사업부_근무실적_현황.html` 파일을 더블클릭해 브라우저로 엽니다.
2. 처음 열면 파일에 내장된 샘플 데이터가 표시됩니다.
3. 우측 상단 **CSV 불러오기** 버튼으로 실제 주간 근무시간 확정 데이터(csv)를 불러오면 화면이 자동으로 갱신됩니다.

인터넷 연결이나 로그인 없이 로컬에서만 동작하며, 불러온 데이터는 외부로 전송되지 않고 브라우저 안에서만 처리됩니다.

## CSV 파일 형식

아래 열 이름이 정확히 일치해야 합니다. (엑셀에서 저장 시 "CSV UTF-8(쉼표로 분리)"로 저장해 주세요. xlsx 파일을 그대로 올리면 오류 안내가 표시됩니다.)

```
No. / 구분 / 년월 / 주차 / 사번 / 이름 / 직급 / 사업장 / 부서명 /
월 / 화 / 수 / 목 / 금 / 토 / 일 /
주확정근무시간 / 평일확정근무시간 / 법정휴게시간 / 비업무_기타 / 비업무 / 비업무(중식)
```

- 요일별 근무시간 및 근무시간 관련 열은 **분 단위**로 입력합니다. (예: 52시간 → 3120)
- `구분`이 **근태예외자**인 인원과 직급이 **상무·전무·부사장·전문위원**인 인원은 자동으로 집계에서 제외됩니다.
- 주차 표기는 `n주차(MM월 DD일~MM월 DD일)` 형식을 따릅니다. 이 값으로 주 단위 추이를 계산합니다.

## 화면 구성

| 영역 | 내용 |
|---|---|
| 52시간 근무자 비율 (도넛) | 이번 주 확정근무시간 52시간 이상 인원 비율·인원수, 전주 대비 증감, 필터 적용 시 사업부 전체 대비 비교 |
| 위험군 KPI | 2주 이상 연속 52시간 이상 근무자 수 |
| 평균 주간 확정근무시간 | 필터 적용 대상의 평균, 52시간 상한 대비 여유시간 |
| 주말 근무 발생 인원 | 토·일 근무가 발생한 인원 수와 관련 부서 수 |
| 주별 52시간 근무자 비율 추이 | 최근 6주 추이 꺾은선 그래프 |
| 부서별 / 사업장별 평균 확정근무시간 | 아래 "부서·사업장 비교 로직" 참고 |
| 주 근무시간대별 인원 분포 | 40~52시간 구간별 도넛 차트, 조각 클릭 시 해당 인원 명단 표시 |
| 평일 요일별 근무시간 편차 | 평일 평균 대비 요일별 편차를 위/아래로 표시 |
| 주말 근무 현황 | 토·일 각각의 근무자 수·평균 근무시간 |
| 위험군 상세 테이블 | 연속 52시간 근무자 명단, 부서, 직급, 연속 주수, 이번 주 근무시간 |

## 필터

상단의 **사업장 전체 / 부서 전체** 드롭다운으로 특정 사업장·부서만 필터링할 수 있습니다.

### 부서·사업장 비교 로직

- 필터를 **전체**로 두면 부서별·사업장별 카드에 전체 부서(또는 전체 사업장) 순위가 막대그래프로 표시됩니다.
- **특정 부서(또는 사업장)를 선택**하면 해당 카드는 "선택한 부서 평균"과 "사업부 전체 평균" 단 두 개의 막대만 비교해서 보여줍니다. (사업부 전체 평균은 필터와 무관하게 항상 전사 기준으로 계산됩니다.)
- 막대 색상은 두 경우 모두 동일한 경고 기준을 따릅니다.
  - 52시간 이상: 코랄(경고)
  - 48시간 이상: 스틸
  - 44시간 이상: 스카이블루
  - 그 미만: 페일
- 점선(세로 틱)은 52시간 기준선을 나타냅니다.

## 데이터 내보내기

| 버튼 | 동작 |
|---|---|
| **완성 파일로 저장** | 현재 불러온 전체 CSV 데이터를 파일 안에 내장한 새로운 독립 HTML을 생성해 다운로드합니다. 생성된 파일은 열자마자 해당 데이터가 자동으로 표시되어 다른 사람에게 그대로 배포할 수 있습니다. 아직 실제 CSV를 불러오지 않은 샘플 상태라면 저장 전 확인 메시지가 표시됩니다. |
| **현재 선택 필터 내용 저장** | 상단에서 선택한 사업장 또는 부서 조건으로 필터링된 데이터만 담아, 위와 동일한 방식의 독립 실행형 HTML로 저장합니다. 사업장·부서가 모두 "전체"인 상태에서는 저장할 수 없으며, 이 경우 하나를 선택하거나 "완성 파일로 저장"을 이용해 주세요. |

두 버튼 모두 결과물은 csv가 아닌 **html 파일**이며, 파일을 열면 저장 시점의 필터링된 데이터가 그대로 재현됩니다.

## 색상·폰트 설정

우측 상단 톱니바퀴(⚙) 아이콘을 누르면 설정 패널이 열립니다.

- 5가지 색상(주요 텍스트/헤더, 보조 텍스트/차트, 차트 보조색, 연한 배경/트랙, 경고색)과 폰트를 자유롭게 변경할 수 있습니다.
- **화면에 바로 적용**: 현재 세션에만 즉시 반영됩니다.
- **style.css용 코드 복사**: 변경한 값을 코드 블록(`:root{...}`)으로 클립보드에 복사합니다. 이 코드를 HTML 파일 상단의 `:root` 블록에 직접 붙여넣으면 영구적으로 반영됩니다.
- **기본값으로 되돌리기**: 최초 색상·폰트로 초기화합니다.

## 오류 및 예외 처리

- 필수 열이 누락된 CSV를 불러오면 어떤 열이 없는지 안내 배너가 표시됩니다.
- 데이터 행이 하나도 없는 CSV를 불러오면 오류 메시지가 표시됩니다.
- xlsx 파일을 그대로 업로드하면 CSV로 다시 저장하라는 안내가 표시됩니다.
- 파일 인코딩은 UTF-8(BOM 포함), UTF-16, EUC-KR/CP949를 자동으로 판별해 읽습니다.
- 선택한 필터 조건에 해당하는 데이터가 없으면 안내 배너가 표시되고 화면이 비워집니다.

## 참고 사항

- 이 파일은 완전히 로컬에서 동작하는 단일 HTML이므로, 회사 인트라넷·이메일 첨부·USB 등 어떤 방식으로도 공유·보관할 수 있습니다.
- 내장된 샘플 데이터는 예시용 가상 인원 정보입니다. 실제 데이터를 불러오기 전까지는 화면 우측 상단에 "샘플 데이터 표시 중"이라는 문구가 표시됩니다.
- 브라우저는 최신 버전의 Chrome, Edge 등을 권장합니다.
