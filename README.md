# 근무시간 준수 리포트 대시보드

원본 근무시간 데이터(A~V열)를 그대로 불러와 계산하는 정적 웹 대시보드입니다. 서버나 로그인 없이 파일만으로 동작합니다.

## 구성

- index.html : 대시보드 화면
- style.css : 디자인 (상단에 색상·폰트 변수 정리)
- script.js : CSV 파싱, 통계 계산, 차트 렌더링 (외부 라이브러리 없이 동작)
- data/timesheet_sample.csv : 가상 샘플 데이터 (330명 × 4주)

## 데이터 불러오기

두 가지 방법이 있습니다.

1. 화면 우측 상단의 "CSV 불러오기" 버튼으로 매주 내보낸 엑셀 파일(CSV로 저장)을 직접 선택합니다. 인터넷 연결이나 로그인 없이 로컬에서 바로 반영되며, 업무용 PC에서 파일을 더블클릭해서 열어도 이 방식은 항상 동작합니다.
2. GitHub Pages로 배포한 경우, data/timesheet_sample.csv를 실제 데이터로 덮어쓰고 커밋하면 접속할 때마다 자동으로 불러옵니다. (파일명은 동일하게 유지해 주세요.)

CSV 열 순서는 원본 그대로 사용하면 됩니다.

```
순번, 근무형태구분, 연월, 주차, 사번, 이름, 직급, 소속사업장, 소속부서명,
월, 화, 수, 목, 금, 토, 일, 주확정근무시간, 평일확정근무시간, 법정휴게시간,
비업무_기타, 비업무, 비업무_중식
```

근태예외자와 임원급(상무·전무·부사장·전문위원)은 자동으로 제외됩니다. "탄력근무제 포함" 체크를 해제하면 탄력근무제 대상자도 뺄 수 있습니다.

## 화면 구성

- 이번 주 52시간 근무 인원 (도넛), 위험군, 평균 근무시간, 주말 근무 인원
- 주별 52시간 근무자 비율 추이 (CSV에 담긴 모든 주차를 자동으로 인식)
- 부서별 평균 근무시간 순위 (52시간 기준선 표시)
- 주 근무시간대별 인원 분포 (40시간 미만 ~ 52시간 이상, 2시간 구간)
- 평일 요일별 근무시간 편차 (평일 평균 대비 위아래로 표시)
- 주말 근무 현황 (평일과 분리)
- 팀별 비업무 자율기입시간 비중
- 위험군 상세 표 (2주 이상 연속 52시간)

필터(사업장/부서/탄력근무제)를 바꾸면 모든 차트가 즉시 재계산됩니다. "이번 주"는 CSV에 담긴 주차 중 가장 최근 주차 기준입니다.

## 색상 · 폰트 설정

우측 상단 톱니바퀴 아이콘을 누르면 패널이 열립니다.

1. 색상 5개(주요 텍스트, 보조 텍스트, 차트 보조색, 연한 배경, 경고색)와 폰트를 원하는 대로 바꿉니다. 바꾸는 즉시 화면에 반영됩니다.
2. 마음에 드는 조합이 나오면 "style.css용 코드 복사"를 누릅니다. 코드가 화면에 표시되고 클립보드에도 복사됩니다.
3. style.css 최상단의 `:root{ ... }` 블록을 복사한 코드로 교체하면, 다음부터는 파일 자체에 색이 고정되어 클로드나 인터넷 없이도 항상 같은 톤으로 열립니다.

## GitHub Pages로 배포하기

1. 이 폴더 전체를 새 GitHub 저장소에 올립니다.
2. 저장소 Settings → Pages → Source에서 Deploy from a branch 선택, 브랜치는 main, 폴더는 / (root)로 저장합니다.
3. https://[아이디].github.io/[저장소명]/ 으로 접속하면 됩니다.

## 로컬에서 미리 보기

파일을 더블클릭해서 열어도 "CSV 불러오기" 버튼으로 파일을 직접 선택하는 방식은 항상 동작합니다. 자동 로딩(샘플 데이터)까지 확인하려면 아래처럼 간단한 로컬 서버를 띄우는 것을 권장합니다.

```
cd 대시보드_폴더
python3 -m http.server 8000
```

이후 브라우저에서 http://localhost:8000 으로 접속합니다.
